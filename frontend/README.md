# Frontend - Chicken Feces Classification

Built with React, TypeScript, Vite, PrimeReact, PrimeFlex, and Tailwind CSS.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Setup environment variables:
   ```bash
   cp .env.example .env
   ```

3. Run development server:
   ```bash
   npm run dev
   ```

## Folder Structure

- `src/components`: Reusable UI components
- `src/pages`: Application pages
- `src/services`: API service calls (Axios)
- `src/hooks`: Custom React hooks
- `src/utils`: Helper functions
- `src/assets`: Images, fonts, etc.
