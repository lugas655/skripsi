# Chicken Feces Classification System

Sistem klasifikasi feses ayam untuk deteksi penyakit menggunakan Vision Transformer (ViT).

## Project Structure

- `frontend/`: React + TypeScript + Vite + PrimeReact + PrimeFlex + Tailwind CSS.
- `backend/`: Node.js + Express + TypeScript + Prisma ORM.
- `ml-service/`: Python FastAPI for ML model inference.

## Prerequisites

- Node.js (v18+)
- Python (v3.9+)
- PostgreSQL (for Prisma)
- Virtual Environment (venv/conda) for ML service
